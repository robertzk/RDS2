Serialize or deserialize an R object according to its RDS2 serialization.

Usage

serialize(object)


deserialize(object)

Arguments

object
ANY. The R object to serialize.

Value

For serialize, the serialized R object. For deserialize, the deserialized R object.

The function attr(object, "RDS2.serialize")$write will be used to perform the serialization and the attr(object, "RDS2.serialize")$read function will be used to perform the deserialization.

Description

Serialize or deserialize an R object according to its RDS2 serialization.
